第2週｜JavaScript 與 jQuery

第3堂：JavaScript 入門

  • JavaScript 基礎語法（變數、資料型別、條件判斷、迴圈）
  • 函式、事件處理（onclick, onchange 等）
  • DOM 操作初步：取得元素與修改內容

JavaScript 範例：

let count = 0;
function add() {
  count++;
  document.getElementById('result').textContent = count;
}

第4堂：jQuery 運用

  • jQuery 語法與簡易選擇器
  • 動態操作 DOM（加入、刪除、修改元素）
  • 事件綁定與動畫效果
  • AJAX 初步認識（簡單取得資料）

jQuery 範例：

$(function(){
  $('#btn').click(function(){
    $('#box').toggle();
  });
});

互動練習區

HTML 練習

請寫出一個包含姓名欄位與送出按鈕的表單：

<form>
  <label>姓名：<input type="text" name="name"></label>
  <input type="submit" value="送出">
</form>

CSS 練習

請為下方區塊加上邊框與內距：

div {
  border: 1px solid #333;
  padding: 10px;
}

JavaScript 練習

請寫一個按鈕，點擊後數字會加一：

<button onclick="add()">加一</button>
<span id="result">0</span>
<script>
let count = 0;
function add() {
  count++;
  document.getElementById('result').textContent = count;
}
</script>
